VBA Lives - Blog

VBA Lives is a blog written for the non-programmer to give a broad and well-ordered overview of VBA in the form of tutorials. There will also be blog posts that supplement the tutorials with code samples, as well as general musings on topics related to Excel, Access, and VBA in general. VBA (Visual Basic for Applications) is a scripting language within Microsoft Office applications, such as Excel and Access, that can be used for automation, web scraping, quality control, and productivity gains. Check out the blog regularly for tutorials, updates, and fun code samples!!
